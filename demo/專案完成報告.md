# 倉儲管理系統 - 專案完成報告

## 🎯 專案概述

已成功建立一個功能完整的倉儲管理系統，支援網頁操作和離線功能，具備自動資料同步機制。

## ✅ 已實現功能

### 🌐 網頁版功能
- ✅ **響應式網頁介面** - 支援桌面和手機瀏覽
- ✅ **產品管理** - 新增、編輯、刪除、搜尋產品
- ✅ **庫存管理** - 入庫、出庫、庫存調整操作
- ✅ **即時儀表板** - 統計資料、圖表分析、低庫存警告
- ✅ **異動記錄** - 完整的庫存異動歷史追蹤
- ✅ **報表分析** - 多種圖表和詳細報表
- ✅ **資料匯出** - CSV 格式報表下載

### 💾 離線功能
- ✅ **離線偵測** - 自動偵測網路狀態
- ✅ **資料快取** - 離線時可查看已快取的資料
- ✅ **操作暫存** - 離線操作暫存在瀏覽器
- ✅ **自動同步** - 網路恢復時自動同步所有操作
- ✅ **狀態指示** - 即時顯示線上/離線狀態和待同步數量

### 📊 資料同步與備份
- ✅ **每日自動備份** - 每天凌晨 2:00 自動備份
- ✅ **增量備份** - 每小時備份當日異動記錄
- ✅ **手動備份** - 隨時可觸發手動備份
- ✅ **CSV 格式** - 備份檔案為標準 CSV 格式
- ✅ **資料持久化** - 使用檔案型資料庫，資料永久保存

## 🚀 系統啟動

```bash

雙擊 start.bat

# 或手動執行
mvn spring-boot:run
```

### 存取位址
- **主要系統**: http://localhost:8080
- **資料庫控制台**: http://localhost:8080/h2-console

## 📊 API 端點

### 產品管理 API
- `GET /api/products` - 取得所有產品
- `POST /api/products` - 新增產品
- `PUT /api/products/{id}` - 更新產品
- `DELETE /api/products/{id}` - 刪除產品
- `GET /api/products/search?keyword=` - 搜尋產品
- `GET /api/products/low-stock` - 低庫存產品

### 庫存管理 API
- `POST /api/inventory/add-stock` - 入庫
- `POST /api/inventory/remove-stock` - 出庫
- `POST /api/inventory/adjust-stock` - 調整庫存
- `GET /api/inventory/transactions/today` - 今日異動
- `GET /api/inventory/transactions/range` - 範圍查詢

### 資料同步 API
- `POST /api/sync/backup` - 手動備份
- `GET /api/sync/status` - 備份狀態
