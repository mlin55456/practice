## 🚀 手機使用方式

### 方法一：連接電腦版 (推薦)

#### 步驟 1: 電腦端設定
```bash
# 啟動伺服器模式
雙擊 start-server.bat

# 系統會顯示存取資訊：
# 本機存取: http://localhost:8080
# 區域網路存取: http://192.168.1.100:8080
```

#### 步驟 2: 手機端存取
1. 確保手機和電腦連接同一個 WiFi
2. 手機瀏覽器開啟區域網路網址
3. 例如：`http://192.168.1.100:8080`
4. 建議加入書籤方便使用

#### 步驟 3: 開始使用
- 點擊左上角 ☰ 開啟選單
- 選擇需要的功能
- 享受完整的倉儲管理體驗

### 方法二：雲端部署 (外網存取)

如果需要在外網使用，可以部署到雲端：

#### Heroku 免費部署
```bash
# 1. 註冊 Heroku 帳號
# 2. 安裝 Heroku CLI
# 3. 部署應用程式
heroku create your-inventory-app
git push heroku main
```

#### 其他雲端選項
- **AWS EC2**: 彈性雲端伺服器
- **Google Cloud**: 穩定可靠
- **Azure**: 企業級服務

## 🆘 手機版常見問題

### Q: 手機無法存取系統？
**A**: 檢查以下項目：
1. 手機和電腦是否在同一 WiFi
2. 電腦防火牆是否允許 8080 埠口
3. IP 位址是否正確

### Q: 手機版介面太小？
**A**: 可以：
1. 使用瀏覽器縮放功能
2. 橫向使用手機
3. 使用平板獲得更好體驗

### Q: 離線功能如何使用？
**A**: 
1. 系統會自動偵測網路狀態
2. 離線時操作會暫存在手機
3. 網路恢復時自動同步

### Q: 如何在手機上備份資料？
**A**:
1. 點擊右上角「手動備份」
2. 或設定自動備份
3. 備份檔案會存在伺服器端
