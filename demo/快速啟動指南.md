# 倉儲管理系統 - 快速啟動指南

## 🚀 一鍵啟動

### Windows 用戶
1. 雙擊 `start.bat` 檔案
2. 等待系統編譯和啟動
3. 瀏覽器開啟 http://localhost:8080

### 手動啟動
```bash
# 1. 建立資料目錄
mkdir data
mkdir data\backup

# 2. 編譯專案
mvn clean compile

# 3. 啟動系統
mvn spring-boot:run
```

## 📱 系統功能

### 🌐 網頁功能
- **儀表板**: http://localhost:8080
  - 即時庫存統計
  - 低庫存警告
  - 今日異動記錄
  - 庫存分類圖表

- **產品管理**: http://localhost:8080/products
  - 新增/編輯/刪除產品
  - 搜尋和篩選功能
  - 庫存調整操作

- **異動記錄**: http://localhost:8080/transactions
  - 查看所有庫存異動
  - 按日期和類型篩選
  - 異動統計分析

- **報表分析**: http://localhost:8080/reports
  - 詳細庫存報表
  - 多種圖表分析
  - 匯出 CSV 功能

### 💾 離線功能
- 網路中斷時自動切換離線模式
- 離線操作會暫存在瀏覽器
- 網路恢復時自動同步所有操作
- 右上角顯示線上/離線狀態

### 📊 自動備份
- **每日備份**: 凌晨 2:00 自動執行
- **增量備份**: 每小時備份當日異動
- **手動備份**: 點擊右上角備份按鈕
- **備份位置**: `data/backup/` 目錄

## 🔧 系統管理

### 資料庫管理
- **H2 控制台**: http://localhost:8080/h2-console
- **連線設定**:
  - JDBC URL: `jdbc:h2:file:./data/inventory`
  - 用戶名: `sa`
  - 密碼: (空白)

### 初始資料
系統首次啟動會自動建立 10 個範例產品，包含：
- 電子產品 (筆記型電腦、滑鼠、鍵盤等)
- 辦公用品 (辦公椅、辦公桌、白板等)
- 辦公設備 (印表機、投影機)
- 文具用品 (文件夾、原子筆)

## 📋 使用流程

### 1. 產品管理
1. 進入「產品管理」頁面
2. 點擊「新增產品」建立產品
3. 填寫產品資訊 (SKU、名稱、類別、數量、價格)
4. 儲存產品

### 2. 庫存操作
1. 在產品列表中點擊「庫存調整」圖示
2. 選擇操作類型：
   - **入庫**: 增加庫存
   - **出庫**: 減少庫存  
   - **調整**: 直接設定新數量
3. 輸入數量和原因
4. 確認調整

### 3. 查看報表
1. 進入「報表分析」頁面
2. 查看各種統計圖表
3. 點擊「匯出報表」下載 CSV 檔案

## ⚠️ 注意事項

### 系統需求
- Java 8 或更高版本
- Apache Maven 3.6+
- 至少 512MB 記憶體
- 100MB 磁碟空間

### 資料安全
- 資料庫檔案位於 `data/inventory.mv.db`
- 備份檔案位於 `data/backup/`
- 建議定期備份整個 `data/` 目錄

### 效能優化
- 系統支援數千筆產品記錄
- 建議定期清理舊的備份檔案
- 可調整備份頻率以節省空間

## 🆘 故障排除

### 啟動失敗
1. 檢查 Java 版本: `java -version`
2. 檢查 Maven 版本: `mvn -version`
3. 確認 8080 埠口未被占用

### 資料遺失
1. 檢查 `data/` 目錄是否存在
2. 從 `data/backup/` 恢復備份檔案
3. 重新啟動系統

### 離線功能異常
1. 清除瀏覽器快取和 localStorage
2. 重新整理頁面
3. 檢查瀏覽器控制台錯誤訊息

## 📞 技術支援

如遇到問題，請檢查：
1. 控制台錯誤訊息
2. 瀏覽器開發者工具
3. `data/backup/` 目錄權限
4. 網路連線狀態
