倉儲管理系統 - 安裝使用說明
====================================

🎯 這是什麼？
一個完整的倉儲管理系統，可以管理產品庫存、記錄異動、產生報表。
支援離線使用，資料自動備份。

📋 系統需求
- Windows 7 或更高版本 (或 Linux/Mac)
- Java 8 或更高版本
- 至少 512MB 可用記憶體
- 100MB 磁碟空間

🚀 安裝步驟

步驟 1: 檢查 Java
開啟命令提示字元，輸入：
java -version

如果顯示版本資訊，表示已安裝 Java
如果顯示錯誤，請先安裝 Java：
https://www.java.com/zh-TW/download/

步驟 2: 解壓檔案
將收到的 ZIP 檔案解壓到任意目錄
例如：C:\倉儲管理系統\

步驟 3: 啟動系統
雙擊 start.bat 檔案
等待系統啟動完成

步驟 4: 開啟瀏覽器
在瀏覽器中開啟：
http://localhost:8080

🎉 開始使用！

📱 主要功能

儀表板 (首頁)
- 查看庫存統計
- 低庫存警告
- 今日異動記錄
- 庫存分類圖表

產品管理
- 新增產品：點擊「新增產品」按鈕
- 編輯產品：點擊產品列表中的編輯圖示
- 刪除產品：點擊刪除圖示
- 搜尋產品：使用搜尋框輸入產品名稱或 SKU

庫存操作
- 入庫：增加庫存數量
- 出庫：減少庫存數量
- 調整：直接設定新的庫存數量
- 在產品列表中點擊「庫存調整」圖示

異動記錄
- 查看所有庫存異動歷史
- 按日期和類型篩選
- 查看異動統計

報表分析
- 各種統計圖表
- 詳細庫存報表
- 匯出 CSV 檔案

🔧 進階功能

資料庫管理
網址：http://localhost:8080/h2-console
連線設定：
- JDBC URL: jdbc:h2:file:./data/inventory
- 用戶名: sa
- 密碼: (空白)

手動備份
點擊右上角的「手動備份」按鈕

離線使用
- 系統支援離線操作
- 右上角會顯示線上/離線狀態
- 離線操作會在網路恢復時自動同步

📁 資料位置

資料庫檔案：data/inventory.mv.db
備份檔案：data/backup/
- products_日期.csv (產品備份)
- transactions_日期.csv (異動備份)

🆘 常見問題

Q: 系統無法啟動？
A: 檢查 Java 是否正確安裝，確認 8080 埠口未被占用

Q: 瀏覽器無法開啟？
A: 確認系統已完全啟動，查看控制台是否有錯誤訊息

Q: 如何備份資料？
A: 複製整個 data 目錄，或使用系統內建的備份功能

Q: 如何恢復資料？
A: 將備份的 data 目錄覆蓋回去，或從 CSV 檔案匯入

Q: 如何更改埠口？
A: 編輯 start.bat，在 java -jar 後加入 --server.port=9090

Q: 多人可以同時使用嗎？
A: 這個版本是單機版，每個人有獨立的資料
   如需多人共享，請使用伺服器版本

📞 技術支援

如果遇到問題：
1. 查看控制台錯誤訊息
2. 檢查 data 目錄權限
3. 確認防火牆設定
4. 重新啟動系統
