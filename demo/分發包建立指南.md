# 📦 倉儲管理系統 - 分發包建立指南

## 🚀 快速建立分發包

### 步驟 1: 編譯專案
```bash
# 停止目前運行的系統 (按 Ctrl+C)
# 然後執行編譯
mvn clean package -DskipTests
```

### 步驟 2: 建立分發目錄
```bash
mkdir distribution
mkdir distribution\data
mkdir distribution\data\backup
```

### 步驟 3: 複製檔案
```bash
# 複製 JAR 檔案
copy target\inventory-web-1.0-SNAPSHOT.jar distribution\inventory-management.jar
```

### 步驟 4: 建立啟動腳本

#### Windows 啟動腳本 (start.bat)
```batch
@echo off
echo 正在啟動倉儲管理系統...
echo.

java -version >nul 2>&1
if %errorlevel% neq 0 (
    echo 錯誤: 未找到 Java，請先安裝 Java 8 或更高版本
    pause
    exit /b 1
)

echo 系統將在 http://localhost:8080 啟動
echo 按 Ctrl+C 停止服務
echo.

java -jar inventory-management.jar
pause
```

#### Linux/Mac 啟動腳本 (start.sh)
```bash
#!/bin/bash
echo "正在啟動倉儲管理系統..."
echo ""

if ! command -v java &> /dev/null; then
    echo "錯誤: 未找到 Java，請先安裝 Java 8 或更高版本"
    exit 1
fi

echo "系統將在 http://localhost:8080 啟動"
echo "按 Ctrl+C 停止服務"
echo ""

java -jar inventory-management.jar
```

### 步驟 5: 建立使用說明 (README.txt)
```
倉儲管理系統 - 使用說明
================================

系統需求：
- Java 8 或更高版本
- 至少 512MB 可用記憶體
- 100MB 磁碟空間

啟動方式：
Windows: 雙擊 start.bat
Linux/Mac: ./start.sh

存取網址：
http://localhost:8080

功能說明：
- 儀表板: 查看庫存統計和圖表
- 產品管理: 新增、編輯、刪除產品
- 庫存異動: 入庫、出庫、調整操作
- 報表分析: 詳細報表和資料匯出

資料庫控制台：
http://localhost:8080/h2-console
連線設定：
JDBC URL: jdbc:h2:file:./data/inventory
用戶名: sa
密碼: (空白)

資料位置：
- 資料庫檔案: data/inventory.mv.db
- 備份檔案: data/backup/

注意事項：
- 首次啟動會自動建立範例資料
- 系統會自動備份資料到 data/backup/ 目錄
- 支援離線操作，網路恢復時自動同步
```

## 📋 完整分發包結構

```
distribution/
├── inventory-management.jar    # 主程式 (約 30MB)
├── start.bat                  # Windows 啟動腳本
├── start.sh                   # Linux/Mac 啟動腳本
├── README.txt                 # 使用說明
└── data/                      # 資料目錄
    └── backup/               # 備份目錄
```

## 📊 分發包特色

- **離線可用** - 不需要網路連線

## 🆘 常見問題

### Q: 如何檢查 Java 是否安裝？
```bash
java -version
```

### Q: 如何更改系統埠口？
在啟動腳本中加入參數：
```bash
java -jar inventory-management.jar --server.port=9090
```

### Q: 如何備份資料？
複製整個 `data` 目錄即可

### Q: 如何恢復資料？
將備份的 `data` 目錄覆蓋回去
